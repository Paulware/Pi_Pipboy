Hardware

This device consists of:
  • Wemos chip
  • 4x20 Display
  • Arduino (mini-pro) control of Dispay
  • 3.7V Lithium Ion battery (14450)
  • 3D printed box
  • On/Off switch

Software

This device will login to the pipboy network and ask for pages. The php file: handleAdmin.php was created to handle communication with this device. handleAdmin.php also uses functions from common.php.
